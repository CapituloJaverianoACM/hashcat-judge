# hashcat-judge
## Node server
- Open terminal
- <code> cd \< Path_to_project_directory \> </code> 
- <code> npm init </code> 
- <code> npm install </code> 

## Run sever
- Open terminal
- <code> cd \< Path_to_project_directory \> </code> 
- <code> npm start </code>
